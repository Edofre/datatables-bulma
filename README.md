# Datatables Bulma

Bulma styling for the jQuery integration of datatables, see following sites for more information:
* https://datatables.net
* http://bulma.io


## Installation

```
npm i datatables-bulma-jquery
```

```javascript
var $ = require( 'jquery' );
require( 'datatables-bulma-jquery' )( window, $ );
```
